# 2026-07-19 — Session 16 — Phase 9 (SARIF / Semgrep ingestion)

## Goal
The pivot's part 1 (ADR-014): `secgraph analyze <path> --sarif results.sarif` renders external SAST
findings (CodeQL, Semgrep, Bandit, …) through the unchanged map + MCP pipeline; the built-in taint
engine becomes the no-SARIF fallback.

## Done
- **`secgraph/ingest/`** (graphify-free): `sarif.py` (SARIF 2.1.0 `results` + `codeFlows` → raw
  findings), `semgrep.py` (native JSON `dataflow_trace`, tolerating the `CliLoc`/`CliCall` variants),
  `normalize.py` (URI normalizer + **root clamp**, severity/CWE/confidence maps, `build_finding`),
  `__init__.py` (`ingest_findings`: parse → build → dedup → deterministic sort; `IngestReport`).
- **`project.py` refactor**: `_annotate_graph_json` now consumes finding **dicts** and binds via a
  structural **ladder** (span → nearest-def → file-node → none, ADR-008); extracted the shared
  `emit_artifacts` tail (both the engine and ingest target it); new `analyze_ingest`. `taint.json`
  gains an `engine:{mode}` block; findings gain `rule_id`/`message`/`hops`/`provenance`/`guard_status`.
- **CLI** `analyze --sarif F --semgrep-json F` (repeatable; any ingest flag skips the engine) + the
  binding report. **viz** empty-neighborhood → full-graph fallback (never a blank map). **MCP**
  `get_path_slice` emits per-hop windows when `hops` exist (closes the ADR-011 deferral).

## Reviewer → fixes (all verified; determinism/quarantine/data-layer/engine-path confirmed clean)
- **HIGH**: the root clamp was a raw string prefix (`/proj` vs `/proj-evil` → escapes) → a hostile
  SARIF could steer a slice read outside the tree. Fixed to a path-boundary check
  (`resolved.relative_to(root)`, re-deriving `cand` so no `..` survives) + regression test.
- **MEDIUM**: no per-result isolation — a wrong-typed field (`locations:["oops"]`, `codeFlows:[123]`,
  `startLine:"NaN"`) aborted the whole ingest (DoS). Fixed: per-result `try/except` → dropped
  `parse-error`, `to_int` coercion; the CLI's catch widened by construction. Regression test.
- **LOW**: SARIF sink vs `hops[-1]` divergence → reconcile (append the authoritative sink);
  file-node heuristic → precise `label == basename`.
