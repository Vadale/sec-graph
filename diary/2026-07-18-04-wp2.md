## Goal
Rules engine (YAML sources/sinks/sanitizers) + intraprocedural flow-sensitive taint over

## Done
- **Closed blocker #2**: `with`/`try`/`match` bodies are now lowered (`with X as v` ->
  alias def `v = X`). In-block defs and returns are captured.
- **Rules engine** (`secgraph/rules/`): typed model, YAML loader (pyyaml, lightweight
  validation), and a matcher that resolves FQNs via a module import map and matches sinks
  by FQN *or* duck-typed method name (`.execute`). Ships `rules/python/flask.yml`.
- **Import map**: `ModuleIR.imports` (local name -> FQN), collected from the tree.
- **Taint engine** (`secgraph/taint/`): forward flow-sensitive worklist over the CFG;
  `expr_taint` with sources / sanitizers (clear) / propagators / over-approx through
  unknown calls; sink detection with confidence + provenance; `run_module` / `scan_project`.
- **CLI**: `secgraph scan <path>` prints findings (graphify-free path).
- Intra fixtures (`tests/fixtures/intra/`): SQLi, command-injection, sanitized.

## Review (reviewer agent) + fixes
Reviewer **confirmed the taint fixpoint is sound and terminating**. Found real defects;
fixed, with regression tests:
- **#1 (HIGH, false negative):** try/except & match were lowered as a FLAT sequential
  list, so a strong-update in a later clause (`try: x=source() except: x='0'`) killed the
  taint before the sink. Now modeled as a **`Branch`** node — CFG alternatives that MERGE
  at a join, so a var reassigned in one clause is unioned, not killed. (My earlier
  "sequential is sound for may-taint" claim was wrong; the reviewer disproved it.)
- **#2 (false negative):** import collection now recurses, so guarded/optional imports
  (`try: from flask import request`) resolve.
- **#4 (false positive):** a parameter/local shadowing an imported name (a param named
  `request`) no longer resolves to `flask.request` — the per-function imap drops shadowed
  names.
- **#5 (determinism):** findings now sort by the full `Finding.key` (a total order);
  verified identical across `PYTHONHASHSEED=0..4` (two sources on one line no longer swap).
