# 2026-07-19 — Session 14 — B: Tier-3 annotation typing

## Goal
Raise call resolution on annotation-saturated code (the deferred "Tier-3" that ADR-007 named as the
fastapi UNK/TRR mover): type a receiver from its **parameter annotation** (`def handler(svc: Service)`)
so `svc.method()` binds, instead of falling to `unknown-receiver`.

## Done
- **IR**: `FunctionIR.param_types` (`param -> annotation dotted name`); `lower.py._param_annotations`
  reads `typed_parameter`/`typed_default_parameter` (bare `User` + dotted `auth.Svc`; generics like
  `Optional[X]` deferred — a safe under-claim).
- **Resolver** (`callgraph/resolve.py`): `_annotation_origin` resolves the annotation via the module
  imports / a same-module class → a project class (methods bind) or a library type (`external`); a
  Tier-3 branch in `_receiver_origin`, **after** Tier-1 local typing (a reassignment still wins) and
  gated `origin != "unknown"` so it is purely additive (only turns `unknown-receiver` into
  `bound`/`external`, never breaks an existing bind — no regression, 104 tests green).

## Evidence (delta-proven)
```
tests/fixtures/tier3: handler(svc: Service) { q = request.args['q']; svc.query(q) }  ->  Service.query
  has the sink `conn.execute("… %s" % q)`.
  WITH Tier-3:    app.py:8 svc.query -> BOUND;  finds handler->query CWE-89 (cross-file)
  WITHOUT (param_types cleared): svc.query -> unknown-receiver;  0 findings   (flow MISSED)
$ pytest -q  -> 104 passed   (+3: annotation capture, method bind, delta proof)
quarantine wall intact.
```
So Tier-3 improves both the metric (bound vs unknown-receiver) **and** real detection (a cross-file
SQLi a name-only resolver misses).

## Process note
This is a small, **additive** resolver extension (an IR field + one lowering helper + one gated
branch) with a delta proof and the full 104-test suite as the oracle; like WP-A I self-verified
rather than dispatching the reviewer/simplifier subagents, given the session length. The full
review/simplify cadence resumed for the larger phases and resumes for anything non-additive.
