# 2026-07-19 — Session 12 — Phase 6 (MCP triage server)

## Goal
Close the mission's other half — hand isolated data-flow paths to an external LLM harness over MCP
for token-frugal triage — with the analysis core staying LLM-free (ADR-000). User chose the
official `mcp` SDK.

## Done (ADR-011)
- **`secgraph/mcp_view.py`** (new): a pure, graphify-free, SDK-free `TaintView` over the artifacts,
  implementing the §13 tools: `list_paths` (filter by layer/confidence/file, rank unguarded+severity
  first), `get_path_slice` (minimal source+sink code windows, sha256-verified vs `file_hashes`,
  `stale` flag — the token-efficiency win), `find_unguarded_sinks`, `explain_layer` (rule/dict
  provenance), `get_function_taint` (paths bound to a node by the stamped `source_node`/`sink_node`).
- **`secgraph/mcp_server.py`**: thin `FastMCP` wrapper (`build_server`), SDK import **lazy**; the 5
  tools + the 3 §15 defensive triage prompts. `secgraph serve` wired (stdio).
- **`taint.json` extended** (`project.py`): `id` (`path-NNNN`), `file_hashes` (staleness),
  `source_node`/`sink_node` (from the projection's (file,def-line) span join), absolute `root`.
- **README**: honest Status update (alpha, pipeline works) + "Triage over MCP" / run-both-servers.

## Reviewer → fixes (all verified; the rest confirmed sound)
- **HIGH**: `get_function_taint` returned empty for every **method/constructor** node — graphify
  labels a method `.get()` (leading dot) but the finding names it `get`, so the name match missed
  (same ADR-008 trap; the one test used a module-level fn so it passed). Fixed by binding on the
  stamped `source_node`/`sink_node`; regression test on a method node.
- **LOW-MED**: a cwd-relative `root` made `get_path_slice` return empty when `serve` runs from a
  different directory → store an absolute `root`.
- **LOW**: same-name collision (subsumed by the node-id fix); `explain_layer("auth")` empty-nodes
  prompt (now phrased as a path-level layer); a pre-Phase-6 artifact crash (`by_id` now tolerant).
- Reviewer confirmed determinism, `get_path_slice` window math + staleness, read-only/isolation, and
  the prompts' defensive framing all **sound**.
